# Relatorio para Entrega - Landing Page FMP DevStart

## 1) O que o site faz
A landing page "FMP DevStart" foi criada para divulgar um minicurso de programacao para iniciantes.
Ela apresenta:
- proposta do curso
- beneficios para o aluno
- conteudo programatico
- formulario de inscricao

Objetivo principal: divulgar o curso e captar interesse/inscricoes.

## 2) Publico-alvo
- estudantes do ensino medio
- universitarios iniciantes
- pessoas que querem entrar na area de tecnologia

## 3) Estrutura da pagina
A pagina foi organizada com HTML semantico:
- `header`: nome do projeto e menu de navegacao
- `main`: secoes principais do conteudo
- `footer`: informacoes finais e link para topo

Secoes implementadas:
- Hero (apresentacao)
- Beneficios
- Conteudo do curso
- Inscricao (formulario)

## 4) Tecnologias utilizadas
- HTML5
- CSS3
- Google Fonts (Poppins e Open Sans)

## 5) Design e padrao visual
Paleta usada:
- principal: `#0057B8`
- secundaria: `#00A6A6`
- fundo: `#F5F8FF`
- texto: `#1F2937`

A pagina tambem possui responsividade para desktop e mobile.

## 6) Funcionalidades
- menu com links internos
- botoes de chamada para acao
- formulario com campos obrigatorios (nome e email)
- rolagem suave entre secoes

## 7) O que foi aprendido
- organizacao de uma landing page completa
- uso de HTML semantico
- estilizar componentes com CSS
- adaptar layout para celular

## 8) Arquivos do projeto
- `index.html`
- `style.css`
- `relatorio-landing-page.md`
- `anexos/landing-desktop.png`
- `anexos/landing-mobile.png`

## 9) Conclusao
O projeto cumpre o objetivo de apresentar e promover um curso para iniciantes, com layout limpo, navegacao clara e formulario para conversao.
