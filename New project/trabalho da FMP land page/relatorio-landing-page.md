# Trabalho - Criação de Landing Page

## 1. Resumo do Projeto
**Nome do projeto:** FMP DevStart

A landing page **FMP DevStart** foi criada para divulgar um minicurso de programação para iniciantes. A página apresenta os benefícios do curso, conteúdo programático e uma chamada para inscrição. O público-alvo são estudantes do ensino médio, universitários iniciantes e pessoas que desejam começar na área de tecnologia. O objetivo principal é **divulgar** o curso e **captar inscrições**.

## 2. Metodologia
Para o desenvolvimento do trabalho, utilizei:
- VS Code para editar o código
- Navegador (Google Chrome) para testes visuais
- DevTools para ajustes de layout e responsividade

Etapas seguidas:
1. Planejamento da ideia e definição do objetivo da página.
2. Organização da estrutura com HTML semântico.
3. Estilização com CSS (cores, tipografia e espaçamento).
4. Ajustes de responsividade para celular e desktop.
5. Revisão final de conteúdo e aparência.

## 3. Estrutura da Página
A página foi organizada com:
- `header`: logo, título e menu simples
- `main`: conteúdo principal dividido em seções
- `footer`: informações finais e direitos autorais

Seções principais:
1. Hero (apresentação do curso)
2. Benefícios (vantagens para o aluno)
3. Conteúdo do curso
4. Chamada para ação (inscrição/contato)

Foram utilizadas `divs` para agrupar elementos e facilitar o alinhamento visual.

## 4. Paleta de Cores
Paleta escolhida:
- **Cor principal:** `#0057B8` (azul)
- **Cor secundária:** `#00A6A6` (verde-água)
- **Cor de fundo:** `#F5F8FF` (cinza claro azulado)
- **Cor de texto:** `#1F2937` (cinza escuro)

A escolha dessas cores buscou transmitir confiança, modernidade e boa leitura.

## 5. Tipografia
Fontes utilizadas:
- **Títulos:** Poppins (entre 32px e 48px)
- **Textos:** Open Sans (entre 16px e 18px)

Essa combinação melhora a hierarquia visual e facilita a leitura do conteúdo.

## 6. Imagens e Recursos
Recursos visuais utilizados:
- Imagem principal de capa (tema tecnologia/estudo), obtida em banco gratuito (ex.: Unsplash/Pexels).
- Ícones de apoio nas seções de benefícios (ex.: Flaticon).

As imagens foram usadas para reforçar o tema da página e tornar o layout mais atrativo.

## 7. Funcionalidades
A landing page possui:
- Botão de chamada para ação ("Quero me inscrever")
- Links de navegação entre seções
- Formulário simples de contato/inscrição (nome e e-mail)

Esses elementos permitem que o usuário interaja com a página e realize a ação principal.

## 8. Considerações Finais
Durante o desenvolvimento, a maior dificuldade foi ajustar espaçamentos e responsividade para diferentes tamanhos de tela. Com isso, aprendi melhor sobre estrutura semântica em HTML, organização de CSS e boas práticas de design visual. Como melhoria futura, eu incluiria validação mais completa de formulário e animações suaves para transições entre seções.
